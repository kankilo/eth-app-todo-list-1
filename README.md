## Get started
Install dependencies
```bash
yarn install
```
Running the project
```bash
yarn start
```


### Run smart contract 
Clone [this repo](https://github.com/dappuniversity/eth-todo-list) to run smart contrack
```bash
git clone https://github.com/dappuniversity/eth-todo-list
```
Install dependencies
```bash
yarn install
```
Start 
```bash
truffle migrate --reset
```
